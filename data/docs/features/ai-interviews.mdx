## Integration Example

<FeaturedExample
  title="Complete Interview Flow"
  description="Handle the entire interview process"
  code={`// 1. Generate interview
const interview = await client.generateInterview({
  projectId: "p123",
  candidateId: "f456"
});

// 2. Conduct interview
const answers = await collectCandidateAnswers(interview.questions);

// 3. Evaluate responses
const evaluation = await client.evaluateInterview({
interviewId: interview.id,
answers: answers
});

// 4. Get hiring recommendation
console.log(evaluation.hiring_recommendation);`}
/>
